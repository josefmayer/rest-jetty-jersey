## RESTful WebService

### Technologies
JSE, Jetty, Jersey


### Steps
##### Build and Run Application
*mvn clean compile exex:java*


##### Build jar, Run jar
*mvn package* <br />
*java -jar target/name-jar-with-dependencies.jar* <br/>
size of jar-file: 5MB


##### Access Application
*http://localhost:2222/home/hello*
